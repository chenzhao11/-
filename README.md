# Recruitment
大二的网站设计课程作业（前端部分）

课程设计选择的主题是求职招聘网站。网站的名字是“新芽招聘”。
网站计划有八个栏目分别为：招聘信息、学长学姐说、行业动态、面试经验、学习充电、内推码、交流论坛、职场说。
做了其中的五个栏目。包括：招聘信息、面试经验、学习充电、学长学姐说、行业动态。

使用bootstrap、html、css，js
